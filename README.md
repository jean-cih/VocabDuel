# VocabDuel 📚

> Простой консольный тренажёр для запоминания английских слов

## 🎯 О проекте

VocabDuel — это минималистичный CLI-тренажёр для расширения словарного запаса. Программа парсит ваш словарь из Markdown-файла и превращает его в интерактивную игру с регулируемой сложностью.

## ✨ Возможности

- 🔄 Два режима перевода: английский → русский и русский → английский
- 🎚️ 4 уровня сложности (разное время на размышление)
- 📖 Автоматический парсинг словарей из Markdown
- ♾️ Бесконечный цикл тренировки
